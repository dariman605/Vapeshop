# 🛍️ VapeShop — Интернет-магазин на Django

Мой первый полноценный пет-проект — интернет-магазин с современным дизайном, корзиной, поиском и авторизацией.

## 🚀 Возможности

- ✅ **Каталог товаров** с карточками и фильтрацией по категориям
- ✅ **Живой поиск** — результаты появляются мгновенно при вводе текста
- ✅ **Корзина покупок** — добавление, удаление, изменение количества без перезагрузки страницы
- ✅ **Авторизация** — регистрация и вход с красивым анимированным фоном
- ✅ **Оформление заказа** — имитация оплаты с очисткой корзины
- ✅ **Адаптивный дизайн** — тёмная тема с анимированными градиентными шариками
- ✅ **Счётчик товаров** в шапке сайта

## 🛠️ Технологии

- **Backend:** Python, Django
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
- **База данных:** SQLite
- **Дизайн:** Glassmorphism, Animated Blobs, Dark Theme


🎨 Дизайн
Проект использует современный тёмный дизайн с анимированными градиентными шариками на фоне (эффект Lava Lamp). Страницы авторизации выполнены в стиле Glassmorphism (эффект матового стекла).
